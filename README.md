## How to install it ?

```javascript
npm install
```

1. You must have a postgres server side : https://www.postgresql.org/download/
2. Duplicate .env.example and change var with yours

```bash
# Server express side
PORT=

# Postgres server side : https://node-postgres.com/apis/client
PGHOST=
PGDATABASE = 
PGPORT=
```

## How to run it ?

```javascript
npm start
```
